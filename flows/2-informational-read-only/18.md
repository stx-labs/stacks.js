# Flow 18 — Quote required STX for a BTC commitment

Before enrolling, a partner or wallet UI computes how much STX must be
paired with a given sats amount under the bond's static ratio. Pure-math
mirror of the contract's `min-ustx-for-sats-amount` read so client and
contract stay bit-for-bit identical (bigint truncating division matches
Clarity uint division).

```ts
import { fetchBond, minUstxForSatsAmount } from '@stacks/bitcoin-staking';

const network = 'mainnet';
const bondIndex = 7;
const intendedSats = 2_500_000_000n; // 25 BTC

const bond = await fetchBond({ bondIndex, network });
if (!bond) throw new Error('bond not configured');

const requiredUstx = minUstxForSatsAmount({
  sats: intendedSats,
  stxValueRatio: bond.stxValueRatio, // ustx per 100 sats (trailing-avg)
  minUstxRatioBps: bond.minUstxRatioBps, // 5% min STX:BTC
});

// Sanity-display values for the UI.
const quote = {
  intendedSats,
  requiredUstx,
  requiredStx: Number(requiredUstx) / 1e6,
  ratioBps: bond.minUstxRatioBps,
};
```

`stxValueRatio` and `minUstxRatioBps` come from the bond record (set by
the Endowment in flow 4). Formula:
`(stxValueRatio * sats / 100) * minUstxRatioBps / 10000`.
