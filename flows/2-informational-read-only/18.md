# Flow 18 — Quote required STX for a BTC commitment

Before enrolling, a partner or wallet UI computes how much STX must be
paired with a given sats amount under the bond's static ratio. Wraps
the contract's `min-ustx-for-sats-amount` read so client and contract
math stay in lockstep.

```ts
import { fetchBond, fetchMinUstxForSats } from '@stacks/bitcoin-staking';

const network = 'mainnet';
const bondIndex = 7;
const intendedSats = 2_500_000_000n; // 25 BTC

const bond = await fetchBond({ bondIndex, network });

const requiredUstx = await fetchMinUstxForSats({
  sats: intendedSats,
  stxValueRatio: bond.stxValueRatio, // ustx per 100 sats (trailing-avg)
  minUstxRatioBps: bond.minUstxRatioBps, // 5% min STX:BTC
  network,
});

// Sanity-display values for the UI.
const quote = {
  intendedSats,
  requiredUstx,
  requiredStx: Number(requiredUstx) / 1e6,
  ratioBps: bond.minUstxRatioBps,
};
```

`stxValueRatio` and `minUstxRatioBps` come from the bond record (set by
the Endowment in flow 4). The on-chain helper is the single source of
truth; do not re-derive locally.
