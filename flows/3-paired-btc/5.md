# Flow 5 — Enroll in a paired-BTC bond

The user pairs BTC with STX in a protocol bond. Three steps: confirm the
caller's allowlist entry covers their intended sats (seeded by the
Endowment in flow 4); construct the L1 timelock address and send BTC to
it; then register on L2.

```ts
import {
  BOND_LENGTH_CYCLES,
  buildLockingScript,
  buildRegisterForBond,
  computeUnlockHeight,
  fetchBondAllowance,
  fetchPoxInfo,
  lockingScriptToP2wsh,
} from '@stacks/bitcoin-staking';
import { broadcastTransaction, fetchNonce, signTransaction } from '@stacks/transactions';

const network = 'mainnet';
const pox = await fetchPoxInfo({ network });
const intendedSats = 2_500_000_000n;  // 25 BTC

// 1. Allowlist check — fails fast if the user isn't approved for this bond
//    or if their cap is below what they want to lock.
const allowance = await fetchBondAllowance({
  bondIndex: bond.index,
  address: user.stxAddress,
  network,
});
if (allowance < intendedSats) {
  throw new Error(`not allowlisted for ${intendedSats} sats on bond ${bond.index} (cap: ${allowance})`);
}

// 2. L1: derive the timelock address. The user funds it from their BTC wallet.
const unlockHeight = computeUnlockHeight({
  firstRewardCycle: bond.firstRewardCycle,
  numCycles: BOND_LENGTH_CYCLES,
  poxInfo: pox,
});
const script = buildLockingScript({
  stxAddress: user.stxAddress,
  unlockHeight,
  earlyExitPubkeys: bond.earlyExitPubkeys,
  earlyExitThreshold: 1,
});
const lockingAddress = lockingScriptToP2wsh(script, 'mainnet');
// → user sends e.g. 25 BTC to lockingAddress out-of-band

// 3. L2: register for the bond.
const tx = buildRegisterForBond({
  bondIndex: bond.index,
  signerManager: 'SP000…USER.signer-manager',
  amountUstx: bond.requiredUstx,
  lockup: { kind: 'btc', outputs: [{ amountSats: intendedSats, txid: btcTxid, outputIndex: 0 }], unlockBytes: script },
  publicKey: user.stxPublicKey,
  fee: 10_000n,
  nonce: await fetchNonce({ address: user.stxAddress, network }),
  network,
});

const signed = signTransaction(tx, user.stxPrivateKey);
await broadcastTransaction({ transaction: signed, network });
```
