# Flow 3 — Unstake (STX-only)

The user sets their position to unlock at the end of the current cycle.
Reverts if called during the prepare phase, so check first.

```ts
import { buildUnstake, fetchPoxInfo, isInPreparePhase } from '@stacks/bitcoin-staking';
import { broadcastTransaction, fetchNonce, signTransaction } from '@stacks/transactions';

const network = 'mainnet';
const pox = await fetchPoxInfo({ network });

// TODO: verify against pox-5.clar that unstake actually reverts during
// the prepare phase before keeping this guard.
const inPrepare = isInPreparePhase({
  burnHeight: pox.currentBurnchainBlockHeight,
  poxInfo: pox,
});
if (inPrepare) throw new Error('cannot unstake during prepare phase');

const tx = buildUnstake({
  publicKey: user.stxPublicKey,
  fee: 10_000n,
  nonce: await fetchNonce({ address: user.stxAddress, network }),
  network,
});

const signed = signTransaction(tx, user.stxPrivateKey);
await broadcastTransaction({ transaction: signed, network });
```
